    Getting Help, Importing Data and Some More Vocabulary

    Getting help An important part of any R package is its documentation. The documentation of any function can be obtained by using ? followed by the name of the function or by using help("function_name"). After the execution of a help command a new page should appear in the lower right window in RStudio. If not, click on the tab Help in the lower right window. The help page contains the title of the function followed by a short description.
